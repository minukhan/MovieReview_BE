# 🎬 CineWall - 영화 리뷰 웹 애플리케이션

CineWall은 최신 영화 리뷰를 공유하고, 추천 영화를 확인할 수 있는 웹 애플리케이션입니다.

## 🚀 주요 기능

- 🎥 영화 목록 및 상세 정보 조회
- 📝 사용자 리뷰 작성 및 조회
- ⭐ 영화 평점 등록
- 🔍 장르별 추천 영화 제공
- 🔒 JWT 기반 인증 및 OAuth2 로그인 지원
